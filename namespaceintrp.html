Sends to all the Interpolators a list of all the points that need to be interpolated onto. Also clears information about data that has already been interpolated, since calling this function triggers a new interpolation.

Called by InterpolationTargetTag::compute_target_points. This is not an Action, but rather a helper function that is called by every Action (LineSegment, Strahlkorper, etc.) that specifies a particular set of points.
